# Quenex - Event Management System

A web-based event management system for Arua City, Uganda.

## Features

- Event organizers can create and manage events
- Attendees can browse and purchase tickets
- Mobile Money integration (MTN/Airtel)
- QR code ticket generation
- Admin dashboard for event approval
- Real-time analytics

## Local Development

1. Clone the repository
2. Create virtual environment: `python -m venv venv`
3. Activate: `venv\Scripts\activate` (Windows) or `source venv/bin/activate` (Mac/Linux)
4. Install dependencies: `pip install -r requirements.txt`
5. Run: `python app.py`
6. Visit: `http://localhost:5000`

## Deployment

This app is configured for deployment on Render.com.
